SLANG → INTENT MAPPING

[Core Champion / Matchup Terms]
Counters / Counterpicks -> champions that perform well against a target champion. Example: "Ahri counters" would give champions that is strong against Ahri, "Who counters Jinx" would give champions that is strong against Jinx and "Shaco counter picks" would give champions that are strong against Shaco.
Gets countered by -> champions that perform well against a target champion (reverse phrasing)
Matchup -> performance comparison between two champions
Hard counter -> strong advantage matchup
Soft counter -> slight or skill-dependent advantage matchup

[Performance & Strength]
OP -> statistically strong / high win rate / meta dominant
Broken -> extremely overpowered
Weak -> underperforming champion
Trash -> underperforming champion
Useless -> underperforming champion
Meta -> current strongest champions or strategies
Off-meta -> uncommon or niche picks
Tier list -> ranking of champions by strength

[Role & Position Terms]
Top -> top lane role
Mid -> mid lane role
Jungle -> jungle role
ADC -> attack damage carry (bottom lane damage role)
Support -> support role
Bot lane -> ADC and support lane
Carry -> high damage champion capable of winning games
Hypercarry -> late-game scaling carry
Peel -> protecting teammates
Engage -> starting fights
Disengage -> escaping or stopping fights

[Gameplay / Strategy Terms]
Build -> items and runes setup
Best build -> most optimal or highest win rate build
Runes -> pre-game stat customization
Pathing -> jungle route optimization
Clear speed -> speed of clearing jungle camps
Scaling -> strength increase over time (late game)
Early game -> early stage power
Mid game -> mid stage power
Late game -> late stage power

[Lane & Farming Terms]
CS -> number of minions killed
Farm -> collecting gold from minions
Wave clear -> speed of clearing minion waves
Push -> forcing minion wave forward
Shove -> forcing minion wave forward
Freeze -> holding wave near tower
Slow push -> gradually building a large wave

[Combat Terms]
Trade -> short damage exchange
All-in -> full commit to kill
Burst -> high damage in short time
DPS -> damage per second
Poke -> long-range chip damage
CC -> crowd control effects (stun, slow, root)
Chain CC -> multiple crowd control effects layered

[Vision & Map Terms]
Ward -> vision placement
Vision -> map visibility
Roam -> leaving lane to assist others
Gank -> surprise attack on lane
Dive -> attacking under tower
Objective -> major map targets (dragon, baron, towers)

[Behavior / Player Terms]
Int -> intentionally dying / feeding (often exaggerated)
Inting -> repeatedly dying or feeding
Feed -> giving kills to enemy
Feeding -> repeatedly giving kills
Snowball -> gaining increasing advantage from early lead
Tilt -> playing worse due to frustration
Smurf -> high-skill player in lower rank

[Query Intent Mappings Examples]
"X counters" -> champions strong against X
"How to beat X" -> counters plus strategy against X
"Best X build" -> optimal items and runes for X
"Is X good" -> meta strength and viability of X
"X tier" -> ranking position of X
"Best champs for role" -> strongest champions for that role
"Synergy with X" -> champions that work well with X
"X guide" -> build, playstyle, and tips for X
